# What I'd prioritize right now

Since you're already building it:

1. Finish onboarding.
2. Make cycle prediction highly reliable.
3. Build symptom + mood tracking.
4. Add AI Coach.
5. Add subscriptions.
6. Launch with 50–100 real testers.
7. Measure retention before adding large features like community.

The biggest risk is not lack of features—it's building too many features before validating that women return to the app every month. Focus first on **retention, trust, and prediction accuracy**. If users consistently come back to log their cycles, everything else becomes much easier to monetize and grow.
